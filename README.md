# Лабораторные работы по дисциплине ПИС за 8 семестр

В этом репозитории содержатся все лабораторные работы по дисциплине “Web‑программирование”, каждая из которых находится в отдельной папке.
Также в репозитории присутствует папка единого Django‑проекта, объединяющего лабораторные работы №2–6.

## Лабораторная работа №1 — Установка и ознакомление с Python

Папка **Lab1** содержит:

- helloworld.py — вывод строки  
- groupmates.py — список студентов  
- filter_students.py — фильтрация по среднему баллу  

Запуск:

```bash
python groupmates.py

```

## Лабораторные работы №2–6 — Django‑проект
Папки **Lab2–Lab6** содержат поэтапные версии проекта, выполненные по методичке.

- Папка Весь Django‑проект содержит полностью собранный рабочий сайт, включающий:
- шаблоны и статику (ЛР2)
- модель Post и админку (ЛР3)
- страницу отдельного поста (ЛР4)
- форму создания поста (ЛР5)
- регистрацию и авторизацию пользователей (ЛР6)

Запуск Django‑проекта
```bash
cd "целый Django-проект"
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
Основные адреса

**Главная страница:**
http://127.0.0.1:8000/ 

**Создание поста:**
http://127.0.0.1:8000/post/new/ 

**Страница поста:**
http://127.0.0.1:8000/post/ 

**Вход:**
http://127.0.0.1:8000/accounts/login/ 

**Регистрация:**
http://127.0.0.1:8000/accounts/register/ 

**Админка:**
http://127.0.0.1:8000/admin/

## Описание проекта
Репозиторий объединяет все лабораторные работы в единую структуру:

* ЛР1 — Python‑скрипты
* ЛР2–ЛР6 — Django‑сайт (блог + авторизация)

Каждая лабораторная находится в отдельной папке, а также собрана в общий рабочий проект.
